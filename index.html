Навыки

Верстаю сайты с использованием HTML5 и CSS3. Знаю JavaScript на уровне применения готовых решений, пишу скрипты для несложных манипуляций. Для стилей использую препроцессор Sass. Знаком с приемами адаптивной и отзывчивой верстки. Придерживаюсь именования классов согласно БЭМ.

Использую Compass для генерации спрайтов, Grunt или Gulp для сборки проекта, Bower для подключения библиотек. Знаком с фреймворками Bootstrap и Foundation на уровне подключения компонентов из исходников (но предпочитаю обходиться без них). Для раскладки по сетке использую Susy.

Интегрирую верстку в Drupal, владею навыками темизации, включая создание темы с нуля на основе фреймворка Omega. Имею опыт разработки собственных модулей для реализации несложного функционала с использованием Drupal API, в том числе решений для кастомизации Drupal Commerce.

Знаю английский на уровне свободного понимания технической литературы и документации и успешного поиска решений проблем на stackoverflow и drupal.org. В течение 5 месяцев посещал курсы разговорного английского от компании Logos, имею сертификат о достижении уровня pre-intermediate.
