Навыки

Кратко / Подробно

Кроссбраузерно верстаю сайты на HTML5 и CSS3. Владею JavaScript и jQuery на уровне манипуляций с DOM и адаптации готовых решений. Использую Sass (SCSS). Знаком с приёмами адаптивной вёрстки, умею и люблю верстать пиксель-перфект. В именовании классов и структуре файлов стилей придерживаюсь методологии БЭМ.

Использую Gulp для автоматизации, Compass для компиляции стилей и генерации спрайтов, Bower для подключения библиотек. Знаком с Bootstrap и Foundation, но предпочитаю собственные решения. Для раскладки по сетке использую Susy. Тестирую вёрстку в Chrome DevTools с помощью плагина PerfectPixel и BrowserSync.

Интегрирую вёрстку в Drupal, владею приёмами темизации, включая создание темы с нуля на основе фреймворка Omega. Знаю PHP на уровне написания кастомных модулей с использованием Drupal API, есть наработки для Drupal Commerce. Работал также с Next Generation CMS, для которой написал пару плагинов.

Работаю с Git-репозиториями, использую Trello для управления проектами по системе Kanban и Grindstone — для учёта времени. Также знаком с Subversion и трекером задач Redmine. Есть опыт работы в окружении Linux (Ubuntu), включая настройку и запуск задач по cron, начальные знания конфигурации Apache и Nginx.

Знаю английский на уровне свободного понимания технической литературы и документации и поиска решений проблем. Прошел пятимесячный курс разговорного английского ESL и получил сертификат о достижении уровня intermediate (сентябрь 2015). Сейчас посещаю курсы Yappi Corporate.

Языки

  • HTML + CSS: семантичная вёрстка с применением новшеств CSS3
  • JavaScript: адаптация готовых решений, jQuery, AJAX, JSON
  • PHP: разработка модулей для CMS Drupal и Next Generation CMS
  • SQL (MySQL, Transact SQL): написание простых запросов

Приёмы

  • кроссбраузерная вёрстка (IE8+) и graceful degradation
  • резиновая и адаптивная вёрстка, mobile-first
  • вёрстка pixel-perfect (с минимальными погрешностями)
  • вёрстка и структурирование кода по методологии БЭМ
  • спрайты, оптимизация кода и ресурсов для production

Окружение

  • препроцессор: Sass (SCSS) + Compass
  • фреймворки: Susy, Bootstrap, Foundation
  • автоматизация: Gulp, Grunt, Bower
  • управление проектами: Trello, Redmine
  • репозитории: Git, SVN
