Навыки

Кратко / Подробно

Кроссбраузерно верстаю сайты на HTML5 и CSS3. Знаю JavaScript на уровне применения готовых решений, пишу скрипты на jQuery. Для стилей использую Sass. Знаком с приёмами адаптивной вёрстки, умею и люблю верстать пиксель-перфект. В именовании классов придерживаюсь методологии БЭМ.

Использую Gulp для сборки проекта, Compass для компиляции стилей и генерации спрайтов, Bower для подключения библиотек. Знаком с фреймворками Bootstrap и Foundation на уровне работы с исходниками (но предпочитаю обходиться без них). Для раскладки по сетке использую Susy.

Интегрирую вёрстку в Drupal, владею приёмами темизации, включая создание темы с нуля на основе фреймворка Omega. Знаю PHP на уровне написания кастомных модулей с использованием Drupal API, есть наработки для Drupal Commerce. Работал также с Next Generation CMS, для которой написал пару плагинов.

Работаю с Git-репозиториями, использую Trello для управления проектами по системе Kanban и Grindstone — для учёта времени. Имею опыт работы с SVN и трекером задач Redmine. Опытный пользователь Linux (Ubuntu): ориентируюсь в настройке, сборке софта из исходников, написании shell-скриптов.

Знаю английский на уровне свободного понимания технической литературы и документации и поиска решений проблем. Прошел пятимесячный курс разговорного английского ESL и получил сертификат о достижении уровня intermediate (сентрябрь 2015). Сейчас посещаю курсы Yappi Corporate.

Языки

  • HTML + CSS: 5 лет опыта — от табличной вёрстки до FlexBox
  • JavaScript: адаптация готовых решений, jQuery, AJAX, JSON
  • PHP: разработка модулей для CMS Drupal и Next Generation CMS
  • SQL (MySQL, T-SQL): написание запросов различной сложности

Приёмы

  • кроссбраузерная вёрстка: IE10+ (при неободимости IE8+)
  • резиновая и адаптивная вёрстка, mobile-first
  • вёрстка pixel-perfect (погрешность до 5px)
  • вёрстка и структурирование кода по методологии БЭМ
  • спрайты, оптимизация кода и ресурсов для production

Окружение

  • препроцессор: Sass (SCSS) + Compass
  • фреймворки: Susy, Bootstrap, Foundation
  • автоматизация: Gulp, Grunt, Bower
  • управление проектами: Trello, Redmine
  • репозитории: Git, SVN
